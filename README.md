# Task 3: Heart Disease Prediction

## Objective
Predict whether a person is at risk of heart disease based on medical data.

## Dataset
Heart Disease UCI Dataset (Kaggle)

## Steps Performed
- Data cleaning (handled missing values)
- Exploratory Data Analysis (EDA)
- Feature encoding (categorical to numeric)
- Model training using Logistic Regression

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- ROC Curve (AUC)

## Key Insights
- Age, chest pain type, and heart rate are important features
- Model shows good predictive performance

## Tools Used
- Python
- Pandas
- Scikit-learn
- Seaborn
- Matplotlib
